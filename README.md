# SCI 论文图表制作工具 (SCI Data Visualization Tool)

## 📌 项目简介

本项目是一个专为科研人员打造的 **轻量级、无需安装、即开即用** 的在线数据可视化工具。它旨在帮助用户快速将实验数据转换为符合 **SCI (Science Citation Index)** 期刊投稿标准的图表。

通过简洁的交互界面，用户可以轻松完成从数据导入、图表类型选择、样式微调到高清导出的一站式流程。

## ✨ 核心功能

### 1. 🔬 SCI 标准风格
- **出版级预设**：默认采用 **白底黑框**（四面封闭）、**刻度朝内**、**无网格线** 的经典学术风格。
- **期刊配色**：内置 **Nature、Science、Lancet、JAMA** 等顶级期刊专属配色方案，同时提供经典、学术、活力等多种通用配色。
- **标准化字体**：支持 **Arial**（默认）、Times New Roman、Helvetica 等常用学术字体。

### 2. 🛠 深度自定义
- **字体与字号**：自由调整图表标题、坐标轴名称、刻度标签的字体和字号，满足不同期刊的排版要求。
- **图例优化**：图例自动放置在绘图区域内部左上角（Inside Top-Left），避免遮挡坐标轴，节省图表空间。
- **数据标签**：支持显示带边框的数据标签，自动适配颜色，清晰直观。
- **细节控制**：支持平滑曲线、显示/隐藏网格线、图例开关等。

### 3. 📂 强大的数据处理
- **多格式支持**：直接拖拽上传 **Excel (.xlsx/.xls)**、**CSV**、**TSV**、**TXT**、**JSON** 文件。
- **一键转置**：内置 **“🔄 转置数据”** 功能，轻松应对行/列颠倒的数据源，无需在 Excel 中手动处理。
- **灵活映射**：可视化选择 X 轴和 Y 轴数据列，支持多系列数据对比。

### 4. 📊 丰富的图表类型
- **折线图 (Line Chart)**：不同系列自动使用不同形状的标记点（圆、方、三角等）。
- **柱状图 (Bar Chart)**：经典的柱状对比。
- **散点图 (Scatter Plot)**：支持多系列散点分布。
- **饼图 (Pie Chart)**：支持百分比显示。
- **热力图 (Heatmap)**：适用于相关性分析或矩阵数据展示。
- **箱线图 (Boxplot)**：用于统计分布分析（自动计算四分位数）。

### 5. 💾 高清导出
- **PNG 导出**：生成高分辨率位图，适合插入 Word/PPT。
- **SVG 导出**：生成矢量图，可直接在 Adobe Illustrator 中二次编辑，无限放大不失真。

## 🚀 快速开始

本项目基于纯静态 HTML/CSS/JS 开发，无需安装 Node.js 或复杂的依赖环境。

### 方法一：直接运行（推荐）

如果你安装了 Python（大多数电脑已预装）：

1.  在项目根目录下打开终端/命令行。
2.  运行命令：
    ```bash
    python -m http.server 8080
    ```
3.  在浏览器中访问：[http://localhost:8080](http://localhost:8080)

### 方法二：双击运行

直接用浏览器打开 `index.html` 文件（注意：某些浏览器的安全策略可能会限制本地文件的文件上传功能，推荐使用方法一）。

## 📖 使用指南

1.  **上传数据**
    *   将你的数据整理为标准的行列格式（第一行为表头）。
    *   点击左侧“数据导入”区域，选择文件或直接拖拽文件上传。
    *   *提示：如果数据行列反了，点击预览区上方的“🔄 转置数据”按钮。*

2.  **预览与检查**
    *   在“数据预览”面板查看数据是否正确解析。

3.  **选择图表**
    *   在“图表类型”面板选择合适的图表（如折线图）。

4.  **数据映射**
    *   **X 轴数据列**：选择作为横坐标的列（如“日期”、“样品名”）。
    *   **Y 轴数据列**：选择一列或多列作为纵坐标数值（支持多选）。

5.  **样式美化**
    *   **图表标题**：输入标题，调整字号。
    *   **轴名称**：输入 X/Y 轴的物理量和单位（如 `Time (s)`、`Intensity (a.u.)`）。
    *   **字体设置**：选择期刊要求的字体（如 Arial），微调字号。
    *   **配色方案**：点击选择 Nature 或 Science 等配色。
    *   **其他**：根据需要开启平滑曲线或数据标签。

6.  **导出图表**
    *   点击“导出 PNG”或“导出 SVG”保存图片到本地。

## 📂 项目结构

```text
G:\google\图制作\
├── index.html          # 主页面（结构与布局）
├── index.css           # 样式表（SCI 风格定义、UI 组件样式）
├── src/                # 源代码目录
│   ├── main.js         # 主逻辑入口（状态管理、事件绑定）
│   ├── data-parser.js  # 数据解析模块（处理 Excel/CSV）
│   ├── chart-renderer.js # 图表渲染核心（ECharts 配置生成、SCI 样式适配）
│   ├── theme.js        # 主题配置（配色方案、基础样式常数）
│   └── exporter.js     # 导出模块（图片下载逻辑）
└── README.md           # 项目说明文档
```

## 🛠 技术栈

*   **HTML5 / CSS3 / ES6 JavaScript**：原生开发，无框架依赖，轻量高效。
*   **ECharts 5**：强大的底层图表渲染引擎。
*   **SheetJS (xlsx)**：Excel 文件解析。
*   **Papa Parse**：CSV 文件解析。
